Help for ValidTill

If you need assistance with reminders, widgets, language settings, or data behavior, use the contact details below.

Contact

Email: contact.oaky@icloud.com

Platform requirements

ValidTill currently requires iOS 17.6 or later.

No account required

The app does not require account creation or sign-in. Core features work without a backend service.

Fast troubleshooting

  • Make sure notifications are allowed in iOS Settings if reminders are missing.
  • Open the app after editing coupons so widgets can refresh shared local data.
  • Check the app language in Profile > Language if text appears unexpected.
  • If a coupon seems incorrect, edit and save it again to refresh reminder logic.

Frequently asked questions

Where is my data stored?

Coupon data, profile settings, and reminder preferences are stored locally on your device. A shared local container is also used so the widget can read the latest snapshot data.

Why didn’t I receive a reminder?

Check whether notification permission is enabled, whether the coupon has a reminder lead time, and whether the coupon is already expired or marked as used.

Does the app support more than one language?

Yes. ValidTill supports English and Simplified Chinese, and you can switch the app language in the Profile tab.

Can I use the app without internet?

Yes. The core coupon-tracking experience is designed to work offline.
